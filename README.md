# Global Block Hardness
**This mod makes breaking blocks harder.**

## How to use:
* Join a world
* Go into serverconfig folder and open mod configuration file
* Set the multiplier to your liking  
  **(1.0 is vanilla behaviour, 2.0 makes blocks 2 times harder to break, 0.5 will make blocks 2 times easier to break and so on...)**
* You can export the config file into the defaultconfigs folder, so every new world you create will use the same settings.
* Enjoy!
